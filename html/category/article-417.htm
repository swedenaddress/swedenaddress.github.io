java笔记之class类定义和名称和屏幕打印（系统输出）

  • 首页

  • 文章分类

  • 正文

1、”public class 类名称{}”一个Java源码只能定义一个public类型的class，并且class名称和文件名要完全一致，进行开发时一般用这个来定义且就只有一个。
2、“class 类名称{}”类名称名字可以和文件名不一致，但是编译后的*.class名称是class定义的类名称而不是文件名，在一个*.java文件里可以定义多个class类名称，编译时也会形成多个对应的*.class文件
3、命名标准：类名称命名时首字母要大写
4、主方法（熟记，面试时可能会考，）：所有程序执行的起点（又叫主类）实例一：

[public] class 类名称 { 	public static void main(String [] args)	{ 		程序的代码由此开始执行 . 	} }

5、屏幕打印(系统输出)：可以直接在命令行中进行内容的显示，语法有两个：
System.out.println()输出之后换行
System.out.print()输出之后不换行
